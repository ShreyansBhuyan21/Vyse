# Vyse
Vyse, is a prototype designed to detect rapid and repetitive patterns of bright light that can trigger epileptic seizures. It provides real-time alerts to the user to help them with their safety, and improves independence in everyday lives.

This project aims to detect rapidly changing light levels to alert the wearer know if they’re currently in an seizure-inducing environment by sending a gentle vibration to their head. 

Even when lights flash faster than is perceptible, it can still be overwhelming and trigger photosensitive epilepsy.

We know that it can be difficult to tell at times whether a room’s lights are flashing or not. If you’ve ever taken a video of a light source and seen it flickering, you know this phenomenon. Being that we can’t see it with the naked eye, it creates a dangerous environment for someone susceptible to flashing lights.

Another challenge was limiting the scope of our project. Making sure we didn’t overextend ourselves and had a realistic timeline was crucial to our success.
